# UCDavis.Killifish
This repository contains helper scripts and C++ being used to explore the genomes of several species of killifish. The project is an ongoing collaboration between the [Catchen](https://catchenlab.life.illinois.edu/) and [Fuller](https://beckyfullerlab.weebly.com/) labs from the University of Urbana-Champaign, and the [Whitehead lab](https://whiteheadresearch.wordpress.com/) from the University of California, Davis.

The code found in `cpp_files/` were largely written for processing `*bam` files during some of the more manual genome assembly curration steps to address the chromosome fusions found in some of the killifish species.

The scripts found in `scripts/` are a mostly a bunch of python scripts used for visualizing some of the general stats when examining the estimated effective population sizes for these fishes.

The README.md will be periodically updated as the project moves forward, with a mature description available once the paper is written and submitted for review.
